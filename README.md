#### Setting up local Git repo
* mkdir examplefolder
* cd examplefolder
* touch README.md
* git init
* git add .
* git commit -m 'first commit'
* git remote add origin https://github.example.git
* git remote -v (to check its pointing to right github repo)
* git push -u origin main
